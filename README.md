# inventario-python
En el presente trabajo se desarrolla una solución básica de programación orientada a la auditoría de inventario y al control de reabastecimiento de artículos. Se hace entonces un programa en Python para saber qué productos necesitan ser repuestos en función del mínimo establecido para cada uno de ellos.
# Auditoría de Inventario - Fundamentos de Programación

## Descripción

Este proyecto desarrolla una solución básica de programación en Python para auditar el inventario de una empresa y determinar qué artículos necesitan ser reabastecidos.

El programa utiliza matrices, funciones, estructuras condicionales y ciclos repetitivos para calcular la cantidad exacta de productos que deben solicitarse según el stock mínimo requerido.

---

## Problema desarrollado

Problema 3 - Auditoría de Inventario

La solución analiza una matriz con información de artículos:
- Código
- Nombre
- Stock actual
- Stock mínimo requerido

El sistema determina:
- Si el artículo necesita reabastecimiento.
- Cuántas unidades deben solicitarse.
